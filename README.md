# DreamMarket

꿈을 분석하고 해석하는 대화형 웹 애플리케이션입니다.

## 🚀 배포

이 프로젝트는 **Vercel**에 배포되어 있습니다.

- **Build**: `npm run build` (Vite)
- **Output**: `dist/`
- **Framework**: Vite (React 19)

## 📦 기술 스택

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite 7.x
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Routing**: Wouter

## 🛠️ 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 📝 주요 기능

- 꿈 분석 및 해석
- 꿈 라이브러리 관리
- 상세 분석 페이지
- 에디터 뷰
